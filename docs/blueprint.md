# **App Name**: FinTrack

## Core Features:

- Category Creation: Create new expense categories with light color indicator badges.
- Expense Tracking: Add new expenses to categories with amount, date, and description.
- Dynamic Summation: Dynamically calculate and display the sum of expenses for each category.
- Budget Setting: Set and edit budget limits/goals for each category.
- Category Merging: Merge multiple categories or move expenses between categories either by total or individually.
- Category and Entry Deletion: Delete categories and entries with automatic sum updates.
- Search and Filter: Implement a search and filter feature to find expenses using keywords or date ranges. LLM tool to detect intent behind a user's prompt or request. (eg, extract all values and categories needed for the prompt)

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a calm, trustworthy feel.
- Background color: Light Indigo (#E8EAF6) for a clean and uncluttered surface.
- Accent color: Deep Purple (#673AB7) to bring balance between calming trust and bold user action.
- Body and headline font: 'PT Sans', a humanist sans-serif providing both modern look and warm personality.
- Use Lucide icons for clear, simple, and easily understandable UX.
- Mobile-first design with a clean and intuitive layout.
- Subtle transitions and animations for a smooth user experience.
